# Annex A: Detailed Use Case Specification Sheets

This annex contains the detailed specification sheet for each of the 11 accepted WP4 use cases. Each sheet is authored by the corresponding UC lead and follows the standardised template introduced in @sec-3-4. The annex is the **authoritative per-UC reference** for the deliverable: where the narrative in @sec-4 summarises a use case, the corresponding sheet in this annex carries the full, field-by-field record on which D4.2 (Implementation Manual) and D4.3 (Validation Report) will build.
