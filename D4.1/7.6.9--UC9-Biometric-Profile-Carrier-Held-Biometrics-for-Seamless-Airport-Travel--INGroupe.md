### UC 9: Biometric Profile - Carrier Held Biometrics - for Seamless Airport Travel
UC9 relies on a unified EUDI Wallet interaction pattern to support airport travel, enabling secure issuance, presentation, and verification of identity and travel‑related credentials. 

The flow is built on standard wallet‑based credential issuance and verifiable‑presentation protocols, combined with proximity interactions (QR, NFC), API‑based integration with airline and airport back‑end systems, and optional biometric verification. 

The implementation uses the following standards and protocols: 
* eIDAS 2.0 / European Digital Identity Wallet Framework, which defines the trust model for issuers, wallets, verifiers, and intermediaries;
* the EUDI Wallet Architecture and Reference Framework (ARF), which provides the architectural baseline for roles, trust relationships, and credential‑exchange patterns
* the Verifiable Credentials (VC) model, used to represent identity, travel entitlements, boarding‑pass data, booking references, and other domain‑specific artefacts.
* Credential issuance follows OpenID4VCI 1.0, while verification and presentation rely on OpenID4VP 1.0, both operating within OAuth 2.0‑based authorization patterns
* User engagement is supported through QR‑code initiation deep links and HTTPS redirects, and NFC‑based proximity interactions, including ISO/IEC 18013‑5 for unsupervised proximity verification of identity and travel evidence. 

UC9 additionally depends on an EUDI Wallet Intermediary Service, which manages issuance and verification flows, performs trust and revocation checks, and exposes APIs to airline, airport, and border‑control systems. 
Integration with sectoral systems uses API‑based connections to airline check‑in, boarding‑eligibility, flight‑status, and gate‑control services. 
Airport‑specific mechanisms include biometric QR / checkpoint‑readable tokens and 1:1 biometric verification, where a live facial capture is compared with the reference linked to the presented credential. 

Together, these standards and protocols enable a coherent, privacy‑preserving, and interoperable airport‑travel journey across the EU.
